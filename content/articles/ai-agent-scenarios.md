## 场景一：日常办公

### 邮件处理
```
@小龙虾
1. 检查今天收到的所有未读邮件
2. 按重要性分为「紧急」「普通」「可忽略」
3. 给「紧急」邮件生成回复草稿
4. 把「可忽略」的归档
```

**效率提升**：从 30 分钟缩短到 3 分钟

### 会议安排
```
@小龙虾
帮我和张三、李四下周三下午找个1小时的空闲时间，
预订3号会议室，发送会议邀请
```

**效率提升**：从 10 分钟（来回沟通）到 30 秒

### 文档处理
```
@小龙虾
把桌面上的 Word 文档转为 Markdown 格式，
统一放到 D:/文档/Markdown/ 目录下
```

## 场景二：开发场景

### 代码审查
- 自动扫描代码仓库的 PR
- 检测潜在的 bug 和代码风格问题
- 生成审查建议

### 文档生成
- 根据代码自动生成 API 文档
- 更新 README 和 CHANGELOG
- 生成项目结构说明

### Bug 修复
```
@小龙虾
检查今天生产环境的错误日志，
找出最频繁的 3 个错误，
尝试给出修复建议
```

---

## 场景三：数据分析

### 日常报表
```
@小龙虾
从上周的销售数据中提取 Top 10 产品，
对比上上周的数据，
生成一份包含增长率的图表报告
```

### 竞品分析
- 自动爬取竞品更新信息
- 汇总成对比表格
- 标注关键变化

---

## 场景四：内容创作

### 社交媒体
- 根据产品更新自动写推文
- 定时发布到多个平台
- 分析互动数据

### 视频剪辑
- 自动识别视频中的关键片段
- 生成字幕和摘要
- 批量处理视频格式转换

---

## 场景五：个人信息管理

### 日程自动化
```
@小龙虾
每天早上 8:30 汇总今天的日程发到微信
每周五下午 5:00 提醒我写周报
```

### 信息搜集
```
@小龙虾
帮我搜集关于「AI Agent 2026年趋势」的最新文章，
整理成带链接和摘要的文档
```

## 场景六：客户服务

- 自动回复常见问题
- 整理客户反馈并分类
- 生成客户需求报告

## 场景七：项目管理

- 自动跟踪项目进度
- 提醒团队成员截止日期
- 生成项目周报和风险报告

## 场景八：学习辅助

- 根据学习材料生成笔记
- 自动搜索补充资料
- 制定学习计划和进度跟踪

## 选择你的第一个场景

如果你是新手，建议从以下场景开始：

| 场景 | 难度 | 推荐理由 |
|------|------|----------|
| 邮件处理 | ⭐ | 最常见、最容易见效 |
| 文件整理 | ⭐ | 操作简单、结果直观 |
| 信息搜索 | ⭐⭐ | 实用性强 |
| 周报生成 | ⭐⭐ | 每周必做，省时明显 |
| 代码审查 | ⭐⭐⭐ | 需要安装技能 |
