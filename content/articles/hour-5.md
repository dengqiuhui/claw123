## 为什么需要配置记忆？

默认的小龙虾是一个「通用助手」，它对你了如指掌——**一无所知**。通过配置记忆和偏好，你可以让小龙虾：

- 🏢 了解你的公司、职位和工作内容
- 📧 记住常用的联系人信息
- 📝 了解你的写作风格和偏好
- ⏰ 适应你的工作节奏和习惯

---

## 第一步：配置个人档案

在小龙虾中发送以下格式的指令：

```
@小龙虾 记住以下信息：
- 我叫张三
- 职位：产品经理
- 公司：XX科技有限公司
- 常用邮箱：zhangsan@example.com
- 工作时间：周一至周五 9:00-18:00
```

你也可以直接在「设置 → 个人档案」中填写。

---

## 第二步：配置工作偏好

### 邮件风格

```
@小龙虾 我的邮件风格偏好：
- 语气：正式但不生硬
- 落款：张三 | 产品经理 | XX科技
- 抄送：默认抄送我的主管李四
```

### 文档格式

```
@小龙虾 我的文档偏好：
- 文件命名格式：日期_项目名_版本号
- 默认保存位置：D:/工作文档/
- 文件格式：优先使用 Markdown
```

---

## 第三步：添加知识库

### 添加项目信息

```
@小龙虾 记住项目信息：
- 项目名：OpenClaw 智能助手
- 负责人：张三
- 截止日期：2026年6月30日
- 团队成员：张三、李四、王五
- 代码仓库：github.com/company/openclaw
```

### 添加常用信息

```
@小龙虾 记住：
- 公司 WiFi：Office-5G / 密码：abc123456
- 会议室预订系统：meeting.company.com
- 打印服务器地址：192.168.1.100
```

---

## 第四步：配置自动化规则

这是一个进阶功能，让小龙虾学会自动处理重复性任务：

```
@小龙虾 设置自动化规则：
规则名：每日邮件总结
触发条件：每天 17:00
执行动作：检查今日未读邮件，生成摘要发送给我

规则名：周报提醒
触发条件：每周五 16:00
执行动作：根据本周已完成任务生成周报草稿
```

---

## 记忆管理技巧

### 查看已存储的记忆

```
@小龙虾 我有哪些记忆？
@小龙虾 关于XX项目的记忆是什么？
```

### 更新和删除记忆

```
@小龙虾 更新我的职位为「高级产品经理」
@小龙虾 删除关于XX的记忆
```

### 记忆分类建议

| 类型 | 内容示例 | 更新频率 |
|------|----------|----------|
| 个人档案 | 姓名、职位、联系方式 | 很少 |
| 项目信息 | 项目名、成员、进度 | 每周 |
| 工作偏好 | 风格、格式、习惯 | 需要时 |
| 临时信息 | 会议密码、WiFi | 偶尔 |
