## 安装前的准备

在开始安装之前，请确认你的电脑满足以下基本要求：

| 要求 | 最低配置 |
|------|----------|
| 操作系统 | Windows 10+ / macOS 12+ |
| 内存 | 8GB RAM |
| 硬盘空间 | 500MB 可用空间 |
| 网络 | 稳定的互联网连接 |

---

## 方式一：QClaw 安装（最简单）

QClaw 是安装门槛最低的选择，不需要下载任何客户端。

### 步骤

1. **打开微信**，扫描 QClaw 官方小程序二维码
2. **授权登录**，允许 QClaw 访问必要权限
3. **首次引导**，跟随小程序内的新手引导完成初始化
4. **开始使用**，在微信对话框里直接向 QClaw 发送指令

> 💡 全程无需安装任何软件，微信扫码 1 分钟搞定！

---

## 方式二：WorkBuddy 安装

### 步骤

1. **访问官网**：打开 [codebuddy.cn/work](https://www.codebuddy.cn/work)
2. **下载客户端**：根据你的操作系统选择对应版本
   - Windows 用户下载 `.exe` 安装包
   - macOS 用户下载 `.dmg` 镜像文件
3. **运行安装程序**：
   - Windows：双击安装包，按提示点击「下一步」
   - macOS：双击 `.dmg` 文件，将应用拖入 `Applications` 文件夹
4. **首次启动**：
   - 打开 WorkBuddy
   - 使用微信/企业微信扫码登录
   - 完成初始设置向导

## 方式三：AutoClaw 安装

### 步骤

1. **访问官网**：[autoglm.zhipuai.cn/autoclaw](https://autoglm.zhipuai.cn/autoclaw/)
2. **下载安装包**：选择 Windows 或 macOS 版本
3. **一键安装**：运行下载的安装程序
4. **首次配置**：
   - 登录智谱账号（或注册新账号）
   - 选择要集成的平台（飞书/浏览器）
   - 完成权限授权

## 首次启动通用检查清单

安装完成后，建议按以下清单检查一遍：

- [ ] ✅ 程序能正常打开，无闪退
- [ ] ✅ 能成功登录账号
- [ ] ✅ 主界面正常显示
- [ ] ✅ 能看到新手引导/教程入口
- [ ] ✅ 设置页面可以正常访问

## 常见安装问题

### Q：安装时被杀毒软件拦截怎么办？

> 部分杀毒软件可能会误报。请将小龙虾程序添加到杀毒软件的白名单中，然后重新安装。

### Q：macOS 提示「无法验证开发者」？

> 进入 **系统偏好设置 → 安全性与隐私 → 通用**，点击「仍要打开」。

### Q：安装后启动闪退？

> 尝试以管理员权限运行程序。如果问题持续，检查系统是否满足最低配置要求。
