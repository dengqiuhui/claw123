## 什么是 SkillHub？

SkillHub 是小龙虾的技能商店，类似于手机的应用商店。每个技能都是一个小龙虾的能力扩展模块。

**安装了「代码助手」技能 → 小龙虾就能审查代码**
**安装了「PPT 制作」技能 → 小龙虾就能生成演示文稿**

## 安装第一个技能：邮件助手

我们以「邮件助手」为例，演示完整的安装和使用流程：

### 安装步骤

1. **打开 SkillHub**：在 IM 中发送 `@小龙虾 打开 SkillHub`
2. **搜索技能**：发送 `搜索邮件助手`
3. **查看详情**：点击技能卡片查看功能介绍
4. **点击安装**：小龙虾会自动下载并配置技能
5. **验证安装**：发送 `@小龙虾 我安装了哪些技能？`

### 首次使用

安装完成后，试试这些指令：

```
@小龙虾 帮我检查今天的未读邮件
@小龙虾 帮我写一封邮件回复客户张总
@小龙虾 把最近的邮件整理成摘要
```

## 技能搭配使用

技能的真正威力在于**组合使用**：

| 场景 | 需要的技能 |
|------|------------|
| 每日工作流 | 邮件助手 + 日历管理 |
| 会议场景 | 会议纪要 + 邮件助手（发送纪要） |
| 项目汇报 | 周报生成 + 文档生成 |
| 信息搜集 | 网页搜索 + 文档生成 |
| 代码开发 | 代码助手 + 文档生成（写注释） |

---

## 🎯 推荐安装顺序

如果你不确定从哪里开始，按以下顺序逐步安装：

1. **邮件助手** — 覆盖最常见的日常需求
2. **网页搜索** — 获取信息的必备技能
3. **会议纪要** — 开会党刚需
4. **文档生成** — 报告和文档自动生成
5. **代码助手** — 程序员必备

> 💡 不要一次性安装太多技能。每装一个，先用几天熟悉它，再装下一个。
