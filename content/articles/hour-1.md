## 开篇：什么是「小龙虾」？

「小龙虾」是 OpenClaw 生态中对基于 OpenClaw 架构的 AI 智能体产品的昵称。它们能替你在电脑上自动执行任务——清理邮件、管理日历、整理文档、搜索信息，甚至帮你写代码。

目前国内已经有超过 10 家厂商推出了自己的「小龙虾」产品，新手最大的困惑就是：**我该选哪一个？**

---

## 头部 6 家小龙虾横向对比

### 🐧 QClaw（微信直连版）
| 项目 | 详情 |
|------|------|
| 出品方 | 腾讯 |
| 安装方式 | 微信扫码即用 |
| 适用人群 | 普通用户、非技术背景 |
| 核心优势 | 零安装、微信生态融合 |
| 不足 | 功能相对基础 |

> 💡 **适合你如果**：你不想折腾安装流程，就想微信扫码立刻用上。

---

### 🐧 WorkBuddy（腾讯龙虾）
| 项目 | 详情 |
|------|------|
| 出品方 | 腾讯 |
| 安装方式 | 客户端下载安装 |
| 适用人群 | 办公族、企业用户 |
| 核心优势 | 企业微信/QQ/飞书全支持 |
| 不足 | 需要一定的配置操作 |

> 💡 **适合你如果**：你日常工作重度依赖企业微信或飞书。

---

### 🔮 AutoClaw（澳龙）
| 项目 | 详情 |
|------|------|
| 出品方 | 智谱（Zhipu AI） |
| 安装方式 | Windows/macOS 一键安装 |
| 适用人群 | 开发者、技术用户 |
| 核心优势 | 浏览器自动化、飞书深度集成 |
| 不足 | 对非技术用户有一定门槛 |

> 💡 **适合你如果**：你经常需要自动化浏览器操作，或者已经是智谱用户。

---

### 🐱 QwenPaw
| 项目 | 详情 |
|------|------|
| 出品方 | 阿里（通义） |
| 安装方式 | 本地/云端双模部署 |
| 适用人群 | 开发者、开源爱好者 |
| 核心优势 | 开源框架、灵活部署 |
| 不足 | 需要一定技术能力 |

> 💡 **适合你如果**：你喜欢开源方案，想要更多的自定义和控制权。

---

### 🐱 CoPaw
| 项目 | 详情 |
|------|------|
| 出品方 | 阿里 |
| 安装方式 | 在线注册使用 |
| 适用人群 | 协作场景用户 |
| 核心优势 | 钉钉/飞书/QQ 多渠道 |
| 不足 | 社区相对较新 |

> 💡 **适合你如果**：你需要团队协作功能，同时使用多种 IM 工具。

---

### 🎵 ArkClaw（火山引擎）
| 项目 | 详情 |
|------|------|
| 出品方 | 字节跳动 |
| 安装方式 | 企业级部署 |
| 适用人群 | 企业用户、大规模部署 |
| 核心优势 | 企业级稳定性、火山引擎生态 |
| 不足 | 偏向企业场景 |

> 💡 **适合你如果**：你在寻找企业级的 AI Agent 解决方案。
